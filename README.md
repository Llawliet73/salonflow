# SalonFlow

SalonFlow is a multi-tenant WhatsApp appointment automation platform for salons.

## Features

- WhatsApp Booking
- Appointment Scheduling
- Reminder Automation
- Customer Management
- Analytics Dashboard

## Tech Stack

- Next.js
- Express.js
- PostgreSQL (Supabase)
- n8n
- Meta WhatsApp Cloud API
